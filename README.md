# 💰 Crypto Faucet Script (Open Source)

A simple and customizable crypto faucet script for developers.

## 🚀 Features
- Wallet-based claim system
- Timer restriction (1 claim per hour)
- Simple frontend (HTML, CSS, JS)
- Beginner-friendly code

## 🎯 Purpose
This project helps developers learn how faucet systems work and build their own Web3 earning platforms.
